# Weather App ☁️

A simple weather app that shows current weather details using OpenWeatherMap API.

## Features
- Search weather by city name
- Shows temperature, description, and humidity
- Simple UI

## Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
